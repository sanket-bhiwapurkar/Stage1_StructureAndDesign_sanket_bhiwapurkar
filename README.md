# Computer Application Department Website

A modern, responsive multi-page website built for a student-driven computing department/community.

## 🌐 Overview

This project showcases a front-end website for a computing department focused on:

- Workshops
- Events
- Student community
- Gallery
- Contact page
- Brand identity

Designed with a clean modern UI using Bootstrap and custom CSS.

## 🚀 Features

- Fully responsive layout
- Reusable hero sections
- Dynamic navbar/footer components
- Events page with countdown UI
- Timeline / Our Journey section
- Gallery with tabbed categories
- Contact page with form layout
- Custom Bootstrap theme colors
- Page loader animation

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Bootstrap 5
